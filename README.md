## ✨ Features

**Fern-Multi V3** is a comprehensive toolkit featuring:

- 🔍 **Network Scanning & Reconnaissance**
- Silent Nmap stealth port scanning
- OS fingerprinting and detection
- Network mapping through TOR

- 🎯 **IP Intelligence**
- IP geolocation tracking
- DNS/NSLookup queries
- IP and phone number data retrieval

- 🛠️ **Security Tools**
- Msfvenom payload generation
- HTTP server deployment
- Media file manipulation utilities
- TOR proxy connectivity management

- 🔐 **Privacy & Anonymity**
- TOR network integration
- Proxy connection management
- Anonymous IP rotation
