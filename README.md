# Fern-Multi

Fern-Multi is a powerful tool that combines several scripts for network scanning, DNS lookups, and local HTTP serving.

## Installation

Run the installer from the repository root:

```bash
bash install.sh
```

## Usage

Start the main menu with:

```bash
python3 Fern-Multi/Fern-Multi-tool.py
```

## Requirements

- Python 3
- bash
- nmap (for scanning features)

## Notes

All internal script references are now relative to the repository path, so the project can run on any machine without hard-coded user directories.
